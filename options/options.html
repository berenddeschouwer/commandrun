Permitted Sites Which sites are permitted to run commands. You can specify specific hosts, eg. localhost.localdomain, or domains, eg. .example.com. A list of sites can be specified.

Allowed Commands What commands are allowed to run. A list of commands can be specified. Important: Sites will be able to add options to the commands, so if you allow /usr/bin/ls, sites can also run /usr/bin/ls --help.
